# Python Fundamentals: Week 1-2 Learning Guide

Master Python basics without overthinking. Python is the only language you need to break into AI.

## What You'll Learn

- Variables, data types, strings
- If/else statements, loops (for, while)
- Functions and return values
- Lists, dictionaries, tuples
- Reading/writing files
- pip and installing packages

## Goal

By end of Week 2: You can write a Python script that takes input, does something with it, and gives output.

That's it.
